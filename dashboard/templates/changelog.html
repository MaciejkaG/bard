Patch 1.5.0 - Change Log

Theme previews!

You can now preview themes before selecting them, simply hover over a theme, and a preview of it will pop up next to your cursor.

Event streams! (stuff's faster)

As I don't specifically like websockets, I've tried to avoid them and I used simple polling to retrieve basically all information on Bard Dashboard from the server. In this update, fetching current track metadata (progress, name, etc.) and queue data is made with SSE (Event Streams).
Because we're no longer using polling, I made the reports way quicker, track metadata synchronisation now happens every 300ms (previously 1s) and queue synchronisation now happens every 500 ms (previously 20s) making it 20 times faster!

We're now less dependant!
